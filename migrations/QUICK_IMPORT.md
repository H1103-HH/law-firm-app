# 📦 导入数据到云数据库

## 已生成的数据文件

在 `migrations/` 目录下已生成以下 JSON 文件：

### 测试数据

1. **lawyers.json** - 5 个律师测试数据
2. **admin.json** - 1 个管理员账号

### 空数据（用户在小程序中产生）

3. **users.json** - 空数组（用户数据）
4. **consultations.json** - 空数组（咨询记录）
5. **viewed_lawyers.json** - 空数组（浏览记录）
6. **saved_cards.json** - 空数组（收藏名片）

---

## 🚀 导入步骤

### 导入律师数据（lawyers）

1. 在云开发控制台，选择集合 **lawyers**
2. 点击右侧的 **"导入数据"** 按钮
3. 选择文件：`migrations/lawyers.json`
4. 点击 **"确定"**
5. 等待导入完成
6. 验证：应该看到 5 条律师记录

### 导入管理员数据（admin）

1. 在云开发控制台，选择集合 **admin**
2. 点击右侧的 **"导入数据"** 按钮
3. 选择文件：`migrations/admin.json`
4. 点击 **"确定"**
5. 等待导入完成
6. 验证：应该看到 1 条管理员记录

### 导入其他集合（可选）

其他集合（users、consultations、viewed_lawyers、saved_cards）目前是空的，可以在小程序中产生数据后再导入。

如果需要导入空数组（为了验证权限设置）：

1. 选择集合
2. 点击 **"导入数据"**
3. 选择对应的 JSON 文件
4. 点击 **"确定"**

---
